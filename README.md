# Data Analyst Portfolio - Vagga Jayalakshmi

I turn raw, messy, and business-critical data into clear insights that support decisions. This portfolio shows end-to-end analytical work across SQL, Python, PySpark, Excel, Power BI, machine learning, NLP, and modern data pipelines.

Across these projects, I focus on the full analyst workflow: understanding the business question, preparing and validating data, writing reproducible analysis, finding patterns, explaining the "so what", and turning the result into recommendations a stakeholder can act on.
